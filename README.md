# Biblioteca

## Descripcion
Este trabajo ha tratado de crear un sistema que gestiona una biblioteca utilizando clases con MVC en java. EL sistema permite gestionar libros y usuarios, además de realizar los préstamos y devoluciones necesarios.

## Como ejecutar el programa
Para ejecutar el programa nos dirigimos al Main y le damos a Run para iniciar el programa, esto hará que el main llame a la vista y comience a ejecutar todo lo que hay en la vista, lo que a su vez llama al controlador para que realice las acciones y maneje los datos.
